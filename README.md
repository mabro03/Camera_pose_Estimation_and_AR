# VisionLens-Calibrator
> **OpenCV를 이용한 카메라 캘리브레이션 및 렌즈 왜곡 보정 도구**

본 프로젝트는 체스보드 패턴을 활용하여 카메라의 내부 파라미터를 추정하고, 광각 렌즈 등에서 발생하는 이미지 왜곡을 수학적으로 보정하는 기능을 제공합니다.

## 📌 주요 기능
- **인터랙티브 이미지 선택**: 비디오에서 캘리브레이션에 적합한 프레임을 사용자가 직접 선택 (Space/Enter 활용).
- **카메라 파라미터 추정**: 초점 거리(f), 주점(c), 왜곡 계수(distortion coefficients) 산출.
- **실시간 왜곡 보정**: 산출된 파라미터를 적용하여 왜곡된 영상을 평면으로 보정(Rectification).

## 📊 카메라 캘리브레이션 결과
실행 결과 산출된 카메라의 내부 파라미터는 다음과 같습니다.

| 파라미터 | 기호 | 결과값 |
| :--- | :---: | :--- |
| **재투영 오차** | $RMSE$ | **[0.182]** |
| **초점 거리 (x)** | $f_x$ | **[432.74]** |
| **초점 거리 (y)** | $f_y$ | **[431.24]** |
| **주점 (x)** | $c_x$ | **[476.06]** |
| **주점 (y)** | $c_y$ | **[288.76]** |

**왜곡 계수 (Distortion Coefficients):**
`k1, k2, p1, p2, k3 = [-0.285, 0.101, -0.0004, 0.0001, -0.018]` (본인의 결과값 입력)
